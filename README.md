# AI 기반 아두이노 자동 분류 시스템

> *카메라로 물체를 인식하고. 아두이노가 서보모터와 LCD를 조작하여 자동 분류하는 시스템

## 프로젝트 소개
>  화학 실험 중 폐기물이 발생하였을 때 무기, 유기, 산, 염기를 나누어 버려야하는데 양 손에 시약을 모두 들고 있어 통을 열지 못하는 문제를 해결하고자 이 프로젝트를 진행하게 되었다.
> 발판을 활용하여 열 수 있지만 이는 발에 화학물이 묻을 위험이 있다.
>
> 따라서 파란 실험 라텍스 장갑을 인식하여 일정 시간 이상 오른쪽/왼쪽 위로 올리면 카메라로 인식하여 그 방향 폐기물 통을 여는 방법으로 이러한 문제를 해결하고자 한다.

## 시스템 구조

```
[카메라]
   │
   ▼
[PC]
  ├─ Vision AI 모델 (MobileNetV2)
  │    └─ 객체 인식 결과 출력
  │
  └─ 시리얼 통신 (USB)
         │
         ▼
    [Arduino]
      ├─ 서보모터
      └─ LCD
```

## 사용 기술 및 부품

### 소프트웨어
| 항목 | 내용 |
|------|------|
| Vision AI | MobileNetV2 / Pytorch / OpenCV|
| 언어 | Python 3.13.12 |
| 아두이노 IDE | v2.x |

### 하드웨어
| 부품 | 수량 | 용도 |
|------|------|------|
| Arduino Uno  | 1 | 메인 컨트롤러 |
| 웹캠 | 1 | 영상 입력 |
| 서보모터 SG90 | 1 | 폐기통 열림/닫힘 조절 |
| LCD | 1 | 통 열림 상태와 시간 표시 |
| 브레드보드 | 1 | 회로 구성 |

## 구현 방법

### 1. Vision AI 모델

- **모델 선택 이유**: (실시간 추론 속도가 빠르고 경량화가 용이하여 선택)
- **데이터셋**: (직접 촬영 / 공개 데이터셋 사용 여부 및 출처)
- **학습 방법**: (Transfer Learning, Fine-tuning 등)


### 2. 아두이노 제어 로직

- 수신된 명령어에 따라 서보모터 각도(0도, 90도, 180도) 제어
